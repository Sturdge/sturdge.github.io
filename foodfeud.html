Food Feud Back

Project Info

  • Genre - Party Game
  • Year - 2015
  • Engine - Unity

Feud Food is a university group project created as part of a Master's degree in game development.

The project was created with the design premise of being a combination of Nintendo's Splatoon and Hudson Soft's Bomberman to create a frantic, fast-paced party game.

The project was subject to several iterative changes. Starting as a grid-based game with basic functionality for painting the arena via walls, to a game with multiple weapons to facilitate the painting mechanics, finally to a game based around dashing into your opponents to splash paint over the arena.

The game features 3 arenas, each with a special mechanic to help swing the match in your favour. If you can get to it first, that is.
