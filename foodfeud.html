Project Info

  • Genre - Party Game
  • Year - 2015
  • Engine - Unity

Feud Food is a university group project created as part of a Master's degree in game development.

The project was created with the design premise of being a combination of Nintendo's Splatoon and Hudson Soft's Bomberman to create a frantic, fast-paced party game.
