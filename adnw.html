Project Info

  • Genre - Puzzle
  • Year - 2022
  • Engine - Godot

Absolutely, Definitely, NOT, Wordle is a small project created with the intention of learning the Godot game engine.

The project was created as a way of learning how to us ethe Godot game engine, testing how easy the engine is to learn and adapt to when compared to Unity, and testing the features Godot has as a game engine.

The core of the project itself was done in a matter of hours, with a significant amount of time spent polishing the game to ensure that it looks and feels good to play, making great use of Godots built-in Tweening functionality..
