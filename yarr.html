Project Info

  • Genre - Score Attack
  • Year - 2015
  • Engine - Unity

Yarr! is an arcade-style score attack game developed as a university project. The game has a tight, yet simple, gameplay loop. The player fires their cannons sequentially at the enemy ship, taking out crates and barrels of resources as well at the enemy crew. By taking out the enemy crew's resources, the player may also receive helpful power-ups!

Due to skill and time constraints, a lot of concepts had to be cut from development. Much of these cut ideas would be implemented if the project was to be developed with an improved skillset. Examples of these ideas include: UI elements to indicate that the cannons are reloading, UI elements to better indicate which cannon the player is currently controlling, and some more interactive elements to add to the gaemplay loop (such letting AI controlled crewmates use cannons).
